## Overview

RustWebApps is a set of HTML-driven calculators centered on PC gaming. It is aimed at people who want quick, browser-based tools for planning, comparing, and checking game-related values without introducing a bulky application layer.

The project is a natural fit for desktop PCs and for users who prefer lightweight web apps for everyday gaming tasks. Since the tools are delivered as HTML, they can be opened with a simple workflow while keeping the experience focused on calculations and practical use.
